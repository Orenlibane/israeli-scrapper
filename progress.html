⚠️

Yad2 TLS bypass in Node.js

Yad2 uses Cloudflare + TLS fingerprinting. The israeli-nadlan-scraper skill uses Python's curl_cffi. For Node.js, need curl-impersonate or a small Python sidecar for Yad2 requests.

⚠️

No Angular skill

Admin UI in Angular has no dedicated skill. Will use standard Angular docs and general patterns. MVP doesn't need the UI — bot commands cover setup.

⚠️

Geocoding / geo proximity

Comparing listings to nearby sold transactions requires geo coordinates. Plan: OpenStreetMap Nominatim API (free, no key) or Google Maps API.

Open: Multi-user vs single-user MVP
